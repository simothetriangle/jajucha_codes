코드 실행하기

1. 코드 업로드

위에 보이는 그림과 같이 upload 아이콘을 클릭하면 현재 프로젝트에 있는 코드를 자주차로 업로드 할 수 있습니다

2. 코드 실행-1

업로드가 완료 된 후 init 버튼을 누르면 starting으로 바뀌면서 카메라를 실행시키게 됩니다. 카메라 실행이 준비되면 ready 문구가 표시되며 아이콘이 초록색으로 변경됩니다

2. 코드 실행-2

ready 버튼을 다시 클릭하게 되면 running으로 문구가 바뀌며 아이콘도 파란색으로 변경됩니다 실질적으로 코드의 main_loop 가 도는 상황이며 이미지를 확인 할 수 있습니다
