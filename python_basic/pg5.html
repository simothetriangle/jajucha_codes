Conditional : 조건문


Theme1 : 조건문- if

if : 만약 ~라면



if 문은 조건에 따라 다른 코드를 실행하도록 하는 중요한 문법입니다. "만약 이 조건이 맞으면, 이 일을 하고, 그렇지 않으면 다른 일을 해"라는 논리입니다.


Theme1 : 조건문- elif

elif : 만약 if 가 아니 라면


"else if"의 줄임말입니다. if 문과 함께 사용하여 여러 조건을 순차적으로 검사할 때 유용합니다. 각 조건이 차례로 검사되며, 처음으로 참인 조건의 코드 블록이 실행됩니다.


Theme1 : Theme1 : 조건문- else

else : if ,elif 아니라면


else 문은 if 문이나 elif 문 다음에 사용되어, 앞의 조건들이 모두 거짓일 때 실행되는 코드를 정의합니다. 쉽게 말해, 모든 조건이 맞지 않을 경우에 실행할 코드를 지정하는 것입니다. else 문은 조건이 필요 없으며, 반드시 마지막에 위치해야 합니다.


Theme1 : 조건문- 예제

a = 200
b = 33

if b > a:
    print("b is greater than a")
elif a == b:
    print("a and b are equal")
else:
    print("a is greater than b")
