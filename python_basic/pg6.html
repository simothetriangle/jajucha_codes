Loop : 반복문


Theme1 : 반복문- while




while 문은 특정 조건이 참인 동안 코드 블록을 반복 실행하는 반복문입니다. 조건이 거짓이 되면 반복이 종료됩니다. while 문은 반복 횟수가 미리 정해져 있지 않고, 조건에 따라 반복을 계속해야 할 때 유용합니다.


break 문은 반복문을 즉시 종료시키는 데 사용되는 제어문입니다. for 루프나 while 루프 내에서 사용될 수 있으며, break 문이 실행되면 현재 반복을 중단하고 반복문 바깥으로 빠져나갑니다.


Theme2 : while - 반복문 예제

i = 1
while i < 6:
    print(i)
    if i == 3:
        break
    i += 1
