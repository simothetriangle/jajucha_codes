문제: 보물 찾기


당신은 고대 유적지에 도착했습니다. 전설에 따르면, 이 유적지에는 보물이 숨겨져 있습니다. 그러나 보물은 간단한 수학적 계산을 통해 발견할 수 있습니다.

  • 유적지의 특이한 특성 상, 보물은 숫자로 이루어진 특정 방정식의 해일 것으로 추정됩니다.
  • 당신에게 주어진 방정식은 다음과 같습니다: ( 𝑎 + 𝑏 ) × 𝑐 + 𝑑
    • a = 10
    • b = 5
    • c = 2.5
    • d = 50

주어진 방정식의 해를 찾고, Check 버튼을 통해서 체점하세요
