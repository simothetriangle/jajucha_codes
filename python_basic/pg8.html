Function : 함수


Theme1 : 함수 - function




def 키워드로 정의하며, 함수 이름과 매개변수를 사용해 호출합니다. 함수는 특정 작업을 수행하는 코드 블록으로 구성되며, 필요에 따라 인수를 전달받아 처리합니다. 작업 결과는 return 키워드를 사용해 반환합니다.


Theme1 : 반복문 - range




파이썬에서 함수 호출은 함수의 이름을 사용하여 이루어지며, 함수 이름 뒤에 괄호를 붙입니다. 이 괄호 안에는 함수가 필요로 하는 인수들을 순서대로 전달할 수 있습니다. 함수 호출이 이루어지면, 함수는 정의된 작업을 수행하고, 필요에 따라 결과값을 반환합니다.


Theme2 : for - 반복문 예제

def my_function():
    return "Hello from a function"

result = my_function()
print(result)  # "Hello from a function"
