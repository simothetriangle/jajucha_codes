Loop : 반복문


Theme1 : 반복문 - for




for 문은 반복문 중 하나로, 주어진 시퀀스(리스트, 튜플, 문자열 등)의 각 요소를 순차적으로 접근하면서 반복 작업을 수행할 때 사용됩니다. for 문은 반복 횟수가 정해져 있을 때 매우 유용합니다.


Theme1 : 반복문 - range




range() 함수는 일련의 연속된 정수를 생성하는 데 사용됩니다. range()는 반복문에서 특히 for 루프와 함께 사용되며, 일정한 간격으로 정수를 생성할 때 유용합니다. 주로 숫자 시퀀스를 만들 때 활용됩니


Theme2 : for - 반복문 예제

for x in range(6):
  if x == 3:
        break
    print(x)
  else:
    print("Finally finished!")
