DATA TYPE : 자료형


세상은 다양한 데이터로 가득합니다. 프로그래밍에서는 데이터를 분류하여 사용하는데, 이를 자료형이라고 합니다. 파이썬에는 크게 3가지의 자료형이 존재 합니다

  • 숫자
  • 문자
  • 불리언

지금부터 자료형의 개념과 종류를 배워보고자 합니다


Theme1 : 자료형- 숫자



숫자형에는 정수 , 실수 , 복소수가 있습니다.

  • 정수는 양수, 음수, 혹은 0이 될 수 있는 숫자입니다.
  • 실수는 소수점이 있는 숫자입니다.
  • 복소수는 실수와 허수 요소를 가진 숫자입니다.

숫자를 아래 코드와 같이 프린트하여 결과를 확인해 보세요

#정수 출력
x1 = 42 
y1 = -10 
print(x1)
print(y1)

################################

#실수 출력
x2 = 3.14
y2 = 1.0
print(x2)
print(y2)

################################

#허수 출력
z = 3+4j
print(z)

Theme1 : 자료형 - 문자


문자열은 단어, 문장 과 같은 일련의 텍스트 데이터를 나타내는데 사용됩니다.

문자열은 작음 따옴표("), 큰 따 옴표("")감싸져있으며, 따옴표 안의 문자들이 문자열로 정의됩니다. 문자를 아래 코드와 같이 프린트하여 결과를 확인해 보세요 

#단어출력
text1  = "hello"
text2 = 'world'
print(text1)
print(text2)

#문장출력
sentense = text1 + text2
print(sentense)

Theme1 : 자료형 - 불리언


불리언(Boolean)은 True와 False의 두 가지 값을 갖는 자료형입니다.

불리언 값은 진실 혹은 거짓을 나타내 기 때문에 프로그래밍에서 조건을 판단할 때 많이 사용됩니다. 불리언을 아래 코드와 같이 프린트하여 결과를 확인해 보세요 

# Boolean 값 할당
is_sunny = True
is_raining = False

print(is_sunny)   # 출력: True
print(is_raining) # 출력: False
